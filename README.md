## Instructions for Build and Use
[(https://youtu.be/1GWEHJawQ30)]
Steps to build and/or run the software:
Install Flutter SDK from the official Flutter website.
Install Android Studio and set up an Android emulator.
Open the project folder in VS Code or Android Studio.
Run flutter pub get to install dependencies.
Start an emulator (either from Android Studio or using flutter emulators --launch <emulator_name>).
Run the app using:
flutter run
Instructions for using the software:
Launch the Android emulator.
Run the application using flutter run.
The app will automatically install and open on the emulator.
If changes are made to the code, press r in the terminal for hot reload to update the app instantly.
